# Vue 3 Contacts CRUD App

## Backend (json-server)
- Run `npm install -g json-server` if not installed
- Start backend: `npm run backend`
- API: http://localhost:3000/contacts

## Frontend (Vite + Vue 3)
- Install dependencies: `npm install`
- Start frontend: `npm run dev`
- App: http://localhost:5173

## Features
- Add, edit, delete, and list contacts
